Installation

Windows / MAC / Linux

Follow the steps below to install and setup the application in Windows PC

  1. Download and unzip the downloaded package and open the /geo-ocean folder to find all the asset and function files. You can also close the github repository

    2. Download


     git clone https://github.com/KrutikaBhatt/geo-ocean
    • geo-ocean/GUI.py - The main python application
      • geo-ocean/plot_functions - Plot functionalities
      • geo-ocean/file_functions - Netcdf file handling
      • geo-ocean/colormaps - Different available colormap
  3. Install the dependencies

    pip install -r requirements.txt

  4. Click and run the GUI.exe file
  5. You can run the application using command line

    python GUI.py

Functionalities

The software is a Installable Cross-platform software that helps processing NETCDF files, visulaize the dataset and provide functionalities for easy statistics and plots

Process NetCDF files

  1. Upload the Netcdf files
  2. View the variable parameters of the satellite file
  3. Get statistical information for the file and parameters
  4. Select colormaps and scales to display the visuallizations
  5. Select the Region of Interest for visulaization







For plotting, we use the matplotlib and cmocean. You can add additional colormaps in plot.py

Statistical Plots

User can generate a number of plots for the timeseries data. The plots available are :

  1. Generate PACF and ACF plots
  2. Generate rolling mean time series plots
  3. View Decomposition of timeseries
  4. Time Distributed series plot
